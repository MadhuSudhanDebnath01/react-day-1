Q1. What is Virtual DOM ?

The virtual Dom or(V DOM) is a programming concept where an ideal or virtual
representation of a UI is kept in memory and synced with "real dom"
by a library such as React Dom. This process is called reconciliation.

Q3. What is difference between Class and Functional Component ?

Ans.

Class Component : -


In react class component is created by creating class inheriting from React.component having render method in it.
it is a stateful component which have state variables in it
In class component render method returns a jsx which is syntatically similar to HTMl
In react Class components life cycle methods are used.

Functional Component : -


Functional component don't have any lifecycle methods they have hooks like UseState, UseEffect.
Complete constructor structure is not required so they are easy to handle.
They do not have state variables.
